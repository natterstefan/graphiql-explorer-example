# graphiql-explorer-example

## Setup

Install dependencies:

```bash
npm install
# or
yarn install
```

Adjust `.env` according to your needs:

```bash
REACT_APP_GQL_URL="http://localhost:3100/graphql"
```

Start the server:

```bash
npm run start
# or
yarn start
```

Your browser will automatically open to <http://localhost:3000> with the
explorer open.
